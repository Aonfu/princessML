Universal YouTube Translator

À propos de nous

Nous sommes quatre étudiants en école d'ingénieur, trois de l'UTBM et un de l'UTC, actuellement en semestre à l'UTSEUS Shanghai. Dans le cadre de l'UV ML01 (Machine Learning), nous avons développé Universal YouTube Translator : une application web qui permet d'obtenir le script traduit par IA dans la langue de votre choix à partir de n'importe quelle vidéo YouTube.

Notre objectif : rendre les contenus YouTube accessibles à tous, quelle que soit la langue.
